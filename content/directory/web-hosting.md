## Static Site & Jamstack Hosting (Best Free Tiers)

- [GitHub Pages](https://pages.github.com/) – **Completely free** hosting for static sites, portfolios, and blogs, served directly from a GitHub repository.
- [Cloudflare Pages](https://pages.cloudflare.com/) – **Free plan for unlimited sites**, teams, and requests, leveraging Cloudflare's massive edge network and CDN.
- [Vercel](https://vercel.com/) – **Free Hobby plan** for personal projects; ideal for Next.js, Nuxt, and static sites with a fast global CDN.
- [Netlify](https://www.netlify.com/) – **Generous free Starter plan** for static sites, basic serverless functions (limited), and continuous deployment.
- [Surge](https://surge.sh/) – **Simple, free** publishing for static web projects via the command line.
- [Firebase Hosting (Google)](https://firebase.google.com/products/hosting) – **Free Spark Plan** for hosting static content, fast CDN, and SSL (part of the larger Firebase ecosystem).
- [NeoCities](https://neocities.org/) – **Free hosting** focused on bringing back the creativity and simplicity of the old web (static only).

## Serverless & App Hosting (Free for Hobby)

- [Render](https://render.com/) – **Free tiers** for static sites, web services (backend), databases, and cron jobs.
- [Railway](https://railway.app/) – **Generous free starter credit** for hosting databases, services, and running applications in the cloud.
- [Fly.io](https://fly.io/) – **Free allowances** for running small full-stack apps close to users with powerful edge features.
- [Deno Deploy](https://deno.com/deploy) – **Free hosting** for JavaScript, TypeScript, and WebAssembly on a global network (serverless functions).
- [Glitch](https://glitch.com/) – **Free hosting** for running full-stack Node.js projects with live editing and collaboration.
- [AWS Amplify](https://aws.amazon.com/amplify/hosting/) – **Free tier** for hosting static web apps and front-ends using the AWS global network.

## Free Traditional Shared Hosting (Limited)

- [InfinityFree](https://infinityfree.com/) – **Completely free** web hosting with cPanel, PHP, MySQL, and no forced ads (slower performance).
- [000webhost (by Hostinger)](https://www.000webhost.com/) – **Free basic shared hosting** with PHP and MySQL (ideal for small WordPress sites).
- [FreeHostia](https://www.freehostia.com/) – **Free Chocolate plan** includes 5 hosted domains, 250MB storage, and 6MB MySQL.
- [ByetHost](https://byet.net/) – **Free hosting** with PHP, MySQL, cPanel, and free technical support.

## Budget VPS & Cloud Providers (Free Credit)

- [Google Cloud (Compute Engine)](https://cloud.google.com/free) – **Always Free tier** includes a micro VM instance (requires credit card for signup).
- [DigitalOcean](https://www.digitalocean.com/) – Offers generous **free credit** for new users to test VPS/Droplets.
- [Linode (Akamai)](https://www.linode.com/pricing/) – Offers substantial **free credit** for new users for testing their cloud computing platform.
- [Vultr](https://www.vultr.com/) – Cloud hosting with competitive pricing and occasional **free credit** for new accounts.
- [Oracle Cloud Infrastructure (OCI)](https://www.oracle.com/cloud/free/) – **Always Free Tier** with a large number of services, including two micro compute instances.

## Code Hosting & Containerization

- [Docker Hub](https://hub.docker.com/) – Free repository service for finding and sharing container images.
- [GitLab Pages](https://docs.gitlab.com/ee/user/project/pages/) – Free static website hosting directly from GitLab repositories.
- [Codeberg Pages](https://codeberg.org/help/pages) – Free hosting for static pages from repositories, focused on open source.
