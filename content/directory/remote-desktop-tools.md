## Cross-Platform Access

- [RustDesk](https://rustdesk.com/) – Free, open-source remote desktop. Self-host optional.
- [AnyDesk](https://anydesk.com/en) – Fast remote access; free for personal use.
- [Chrome Remote Desktop](https://remotedesktop.google.com/) – Simple remote access via Chrome.

## Meeting & Screen Control

- [Microsoft Teams](https://www.microsoft.com/en/microsoft-teams/group-chat-software) – Screen share + remote control during calls.
- [Zoom](https://zoom.us/) – Screen share with remote control.
- [Google Meet](https://meet.google.com/) – Screen share; limited remote control via extensions.

## File Transfer & Security

- [FileZilla](https://filezilla-project.org/) – FTP/SFTP for moving files securely.
- [OpenSSH](https://www.openssh.com/) – Secure shell for server access (CLI).
