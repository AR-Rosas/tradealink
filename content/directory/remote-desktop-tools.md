## General Use & Cross-Platform

- [AnyDesk](https://anydesk.com/) – Fast, secure, and lightweight remote desktop application widely used for personal and professional access (free for personal use).
- [TeamViewer](https://www.teamviewer.com/) – Industry-leading tool for remote control, desktop sharing, and file transfer (free for personal use).
- [RustDesk](https://rustdesk.com/) – Open-source remote desktop software written in Rust, offering a simple alternative to TeamViewer (free for self-hosting).
- [Splashtop Business Access](https://www.splashtop.com/business-access) – Fast, high-performance remote access solution (paid, but highly reliable).
- [Zoho Assist](https://www.zoho.com/assist/) – Cloud-based remote support and access software (free limited edition available).
- [Remmina](https://remmina.org/) – Free, open-source remote desktop client for Linux and other Unix-like systems, supporting multiple protocols.
- [NoMachine](https://www.nomachine.com/) – Fast, secure remote access designed for performance-intensive tasks like video editing and gaming (free for personal use).
- [BeyondTrust Remote Support](https://www.beyondtrust.com/remote-support) – Enterprise-level remote support solution (paid, but common in large IT environments).

## OS-Specific Built-in Tools

- [Microsoft Remote Desktop (RDP)](https://www.microsoft.com/en-us/p/microsoft-remote-desktop/9wzdncrfj3ps) – Built-in protocol for Windows Pro/Enterprise and a client for Mac/iOS/Android for accessing Windows machines.
- [Google Chrome Remote Desktop](https://remotedesktop.google.com/) – Simple, secure, web-based remote access accessible via the Chrome browser or app (completely free).
- [Apple Screen Sharing (VNC)](https://support.apple.com/guide/mac-help/share-your-screen-using-messages-mh30335/mac) – Built-in VNC-based tool for remote access between macOS devices.
- [VNC Connect (RealVNC)](https://www.realvnc.com/en/connect/) – Professional, robust VNC solution for cross-platform access (free for personal use).
- [LogMeIn Central](https://www.logmein.com/central) – Cloud-based remote access and endpoint management (paid, but very common for IT administration).

## Command Line & SSH Access

- [PuTTY](https://www.putty.org/) – Free and open-source SSH, Telnet, and Rlogin client for Windows (essential for developers/sysadmins).
- [MobaXterm](https://mobaxterm.mobatek.net/) – Enhanced terminal for Windows with SSH, X11 server, and RDP/VNC client built-in (free Home Edition).
- [Termius](https://termius.com/) – Modern SSH client with cloud sync and mobile support (generous free tier).
- [Xshell](https://www.netsarang.com/en/xshell/) – Powerful SSH/Telnet client for Windows with advanced features (free for home and school use).
- [Windows Terminal](https://www.microsoft.com/en-us/p/windows-terminal/9n0dx20spt4h) – Microsoft's modern, multi-tab terminal application for Command Prompt, PowerShell, and WSL.

## Collaboration & Screen Sharing

- [Discord Screenshare](https://discord.com/) – Free, high-quality screen sharing and voice chat, often used for casual support and co-working.
- [Zoom Meetings](https://zoom.us/) – Primary video conferencing tool that includes a dedicated screen-sharing control feature.
- [Google Meet](https://meet.google.com/) – Collaboration tool that allows full-screen or window-specific sharing.
- [Skype](https://www.skype.com/en/) – Free video call and screen sharing application.
- [Slack Huddles / Calls](https://slack.com/) – Built-in video calling and screen sharing for teams within the Slack workspace.
