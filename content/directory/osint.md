## General Search & Archives

- [Google Dorks](https://www.exploit-db.com/google-hacking-database) – Advanced search operators for finding specific information (e.g., login pages, files, error messages).
- [Brave Search](https://search.brave.com/) – Privacy-focused search engine with its own independent index.
- [Yandex](https://yandex.com/) – Popular search engine, often useful for finding information in non-Western regions or image reverse search.
- [DuckDuckGo](https://duckduckgo.com/) – Privacy-focused search engine with handy "bangs" for quick site searches.
- [The Wayback Machine](https://archive.org/web/) – Digital archive of the World Wide Web; view historical versions of websites.
- [Archive.is](https://archive.is/) – Tool for taking snapshots of web pages and preserving them as they appear.
- [CachedView](https://cachedview.com/) – Quickly check cached versions of a website from Google, Wayback Machine, etc.
- [Shodan](https://www.shodan.io/) – Search engine for internet-connected devices (IoT, servers, cameras) rather than content.
- [Censys](https://censys.io/) – Search engine to find information about any device exposed on the Internet.
- [URLScan.io](https://urlscan.io/) – Sandbox to analyze and scan URLs for malicious activity, showing components and links.

## People & Identity Search

- [Pipl](https://pipl.com/) – Identity resolution engine focused on finding profiles, contact info, and public records.
- [Social Catfish](https://socialcatfish.com/) – Reverse search by name, email, phone, and image (limited free search).
- [NameChk](https://namechk.com/) – Checks username availability across hundreds of social media platforms and websites.
- [Holehe](https://github.com/megadose/holehe) – Tool to check if an email is registered to various services (via password reset pages).
- [Epieos](https://epieos.com/) – OSINT tool focused on searching email addresses against public data sources.
- [Hunter.io](https://hunter.io/) – Finds email addresses associated with a domain (limited free searches).
- [DeHashed](https://dehashed.com/) – Search engine for security breaches and leaked databases.
- [Have I Been Pwned](https://haveibeenpwned.com/) – Checks if an email address has been compromised in a data breach.
- [OSINT Framework](https://osintframework.com/) – A comprehensive map of OSINT resources organized by category.
- [WhatsMyName](https://whatsmyname.app/) – Checks username activity across multiple websites.
- [KnowEm](https://knowem.com/) – Checks for the use of a username across 500+ popular social media sites.

## Social Media Analysis

- [Tinfoleak (Twitter/X)](https://tinfoleak.com/) – Comprehensive tool for gathering intelligence from Twitter/X profiles (requires API access).
- [Spoonbill](https://spoonbill.io/) – See recent changes to Twitter/X profiles (bio, name) made by people you follow.
- [Instaloader](https://instaloader.xyz/) – Python library to download posts, profile info, and metadata from Instagram.
- [TweetDeck (X Pro)](https://tweetdeck.twitter.com/) – Advanced dashboard for tracking real-time searches and lists on Twitter/X.
- [Facepager](https://facepager.eu/) – Application for fetching public data from Facebook, Twitter, and other platforms.
- [Stalkface (Facebook)](https://stalkface.com/) – Unofficial tool for finding hidden public data on Facebook profiles.
- [Reddit Search](https://camas.unddit.com/) – Un-delete tool for searching deleted Reddit comments and posts.
- [Subreddit Analysis](https://subredditstats.com/) – Provides historical data, growth, and user demographics for subreddits.

## Domain & Infrastructure Tools

- [Whois](https://who.is/) – Look up domain registration records, owner details, and contact information.
- [ViewDNS.info](https://viewdns.info/) – Offers multiple DNS tools, reverse IP lookup, and domain history.
- [Reverse IP Lookup](https://reverseiplookup.com/) – See what other domains are hosted on the same IP address.
- [SecurityTrails](https://securitytrails.com/) – Searches domain and IP history, including historical DNS records.
- [Wappalyzer](https://www.wappalyzer.com/) – Browser extension to identify technologies used on websites (CMS, framework, analytics).
- [BuiltWith](https://builtwith.com/) – Detects website technology stack, tracking tools, and advertising networks.
- [DNSdumpster](https://dnsdumpster.com/) – Free domain research tool to discover hosts, DNS, and open ports.
- [Sublist3r](https://github.com/aboul3la/Sublist3r) – Python script for enumerating subdomains of a target website.

## Geospatial Analysis (GEOINT)

- [Google Maps](https://maps.google.com/) – Standard mapping for satellite imagery, street view, and location data.
- [Google Earth Pro](https://www.google.com/earth/versions/) – Free desktop application for advanced geospatial analysis and historical imagery.
- [OpenStreetMap (OSM)](https://www.openstreetmap.org/) – Free, editable map of the world built by a community of mappers.
- [Wikimapia](http://wikimapia.org/) – Combines Google Maps with a Wikipedia-like system for user-added descriptions.
- [Flightradar24](https://www.flightradar24.com/) – Real-time flight tracking information for commercial aircraft.
- [MarineTraffic](https://www.marinetraffic.com/) – Real-time position data for ships and vessels worldwide.
- [SunCalc](https://suncalc.org/) – Shows sun and moon positions for a specific time and location (useful for image verification).
- [PeakFinder](https://www.peakfinder.org/) – Helps identify mountain peaks in photos based on location and perspective.
- [Webcams](https://www.insecam.org/) – Search engine for unsecured surveillance cameras worldwide.

## Image & Video Analysis

- [Google Reverse Image Search](https://images.google.com/) – Upload an image to find its source, other copies, and related images.
- [TinEye](https://tineye.com/) – Reverse image search engine that tracks where an image appears online.
- [Yandex Image Search](https://yandex.com/images/) – Often finds different results than Google, particularly for non-Western content.
- [ExifTool](https://exiftool.org/) – Desktop tool for reading, writing, and editing metadata in images, videos, and documents.
- [Jeffrey's Exif Viewer](http://exif.regex.info/exif.cgi) – Online tool to quickly check image metadata (EXIF data).
- [Forensically](https://29a.ch/photo-forensics/#forensic-magnifier) – Online image analysis tool for detecting photo manipulation and noise analysis.
- [YouTube Geofind](https://www.youtube.com/channel/UC-K-7Tq5pPz9n2Lg9y5iV7g) – Unofficial tool for searching YouTube videos by location and date.
- [InVID Verification Tool](https://www.invid-project.eu/) – Browser extension for verifying video and image content (checking frames, metadata).

## Deep & Dark Web Resources

- [Tor Browser](https://www.torproject.org/download/) – Free software for enabling anonymous communication and accessing the dark web.
- [OnionSearch](https://github.com/toxic-j/OnionSearch) – Tool for searching the dark web's .onion sites.
- [The Dark Eye (Dark Search Engine)](https://darkeye.cc/) – Search engine indexing content from the dark web.
- [Subreddit: r/deepweb](https://www.reddit.com/r/deepweb/) – Community discussion about dark web topics and links.

## Documentation & Learning

- [The OSINT Curious Project](https://osintcurious.com/) – Provides blogs, webinars, and tips for learning OSINT techniques.
- [Bellingcat Toolkit](https://www.bellingcat.com/resources/) – Guides and resources for open-source investigation methods.
- [r/OSINT](https://www.reddit.com/r/OSINT/) – Active Reddit community for sharing tools, cases, and advice.
- [Trace Labs](https://www.tracelabs.org/) – Nonprofit organization that crowdsources OSINT to help find missing persons.
