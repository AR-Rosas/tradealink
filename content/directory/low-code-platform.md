## Web App Builders

- [Bubble](https://bubble.io/) – Powerful visual web app creator with workflows and database support.
- [Webflow](https://webflow.com/) – Visual website builder that outputs clean code and scalable hosting.
- [Softr](https://www.softr.io/) – Create web apps from Airtable or Google Sheets without code.
- [Bildr](https://www.bildr.com/) – Flexibly build web apps and sites visually.
- [WeWeb](https://www.weweb.io/) – Visual builder for dynamic web applications.
- [Carrd](https://carrd.co/) – Super-simple one-page site builder, often free.
- [Hostinger](https://www.hostinger.com/website-builder) – Affordable, beginner-friendly web creation tool.

## Mobile App Builders

- [Adalo](https://www.adalo.com/) – Build mobile and web apps with drag-and-drop; beginner-friendly.
- [Glide](https://www.glideapps.com/) – Turn spreadsheets into polished apps quickly.
- [Thunkable](https://thunkable.com/) – No-code platform for native mobile apps on iOS and Android.
- [AppSheet](https://www.appsheet.com/) – Google-backed tool to build apps from data sources like Sheets.
- [Momen](https://momen.co/) – Easy no-code tool for apps with AI features.

## Internal Tools & Dashboards

- [Retool](https://retool.com/) – Build internal dashboards with drag-and-drop and optional code.
- [Appsmith](https://www.appsmith.com/) – Open-source low-code tool for internal apps and dashboards.
- [Budibase](https://budibase.com/) – Build internal business apps quickly with automation.
- [Internal.io](https://internal.io/) – No-code builder focused on internal tooling (free tier available).
- [Zoho Creator](https://www.zoho.com/creator/) – Build custom business apps with low-coding.
- [Quickbase](https://www.quickbase.com/) – Build custom workflow apps with minimal code.
- [Kissflow](https://kissflow.com/) – Low-code tool for business process apps and automations.

## Backend & Automation

- [Zapier](https://zapier.com/) – Connect apps and automate tasks without code.
- [Make](https://www.make.com/) – Visual automation builder with robust logic.
- [Firebase](https://firebase.google.com/) – Free tier backend with auth, database, and hosting tools.
- [Supabase](https://supabase.com/) – Open-source backend as a service with realtime Postgres.
- [Xano](https://xano.com/) – No-code scalable backend with automation logic.
- [Backendless](https://backendless.com/) – Visual backend builder with free tier options.
- [Hasura](https://hasura.io/) – Instant GraphQL over your database, minimal code needed.
- [n8n](https://n8n.io/) – Open-source workflow automation you can self-host.
- [IFTTT](https://ifttt.com/) – Simple triggers connecting apps and services.
- [Parabola](https://parabola.io/) – Automate and transform data without coding.

## AI & Enterprise

- [Microsoft Power Apps](https://powerapps.microsoft.com/) – Low-code platform tied into Microsoft 365.
- [OutSystems](https://www.outsystems.com/) – Enterprise low-code for complex apps (free trial).
- [Appian](https://www.appian.com/) – Low-code process automation and case management.
- [Voiceflow](https://www.voiceflow.com/) – Build voice and chat experiences without code.
- [Claude / Anthropic](https://www.anthropic.com/) – Build interactive AI apps through conversational prompts.
- [GitHub Spark](https://github.com/) – Tools from GitHub to help build full-stack apps with low-code helpers.
