## Code Hosting & Version Control

- [GitHub](https://github.com) – Free git hosting and collaboration for code projects.
- [GitLab](https://gitlab.com) – Free Git repository hosting with CI/CD features.
- [Bitbucket](https://bitbucket.org) – Free git hosting with pipelines for small teams.

## Online Editors & IDEs

- [CodePen](https://codepen.io) – Free online editor for front-end snippets and experiments.
- [JSFiddle](https://jsfiddle.net) – Free playground for HTML/CSS/JS prototyping.
- [StackBlitz](https://stackblitz.com) – Free cloud IDE for web apps with instant dev environment.
- [Replit](https://replit.com) – Free online coding environment with multi-language support.
- [Glitch](https://glitch.com) – Free platform to build and remix full-stack web apps.
- [VS Code](https://visualstudio.microsoft.com/vs-code/) – Free powerful code editor beloved by coders.

## Runtimes & Languages

- [Node.js](https://nodejs.org) – Free JavaScript runtime for building APIs and servers.
- [Deno](https://deno.land) – Secure modern runtime for JavaScript and TypeScript.
- [Python](https://www.python.org) – Free Python language for backend development.

## Backend Frameworks

- [Express.js](https://expressjs.com) – Free minimal web framework for Node.js APIs.
- [Fastify](https://fastify.dev) – Free fast web framework for Node.js.

## Frontend Frameworks

- [Next.js](https://nextjs.org) – Free React framework for hybrid static & server-rendered apps.
- [Nuxt](https://nuxt.com) – Free Vue framework for SSR & static sites.
- [Remix](https://remix.run) – Free full-stack React framework with web standards.
- [Svelte](https://svelte.dev) – Free framework for reactive UI with minimal code.

## Hosting & Deployment

- [Vercel](https://vercel.com) – Free hosting with automatic deployments for front-end projects.
- [Netlify](https://netlify.com) – Free static site hosting with serverless functions.
- [Surge](https://surge.sh) – Free simple static hosting via CLI.
- [Render](https://render.com) – Free tier for web services and APIs.
- [Railway](https://railway.app) – Free cloud platform for apps and databases.
- [Fly.io](https://fly.io) – Free edge hosting for apps close to users.

## Databases & ORMs

- [PostgreSQL](https://postgresql.org) – Free robust SQL database for apps.
- [Redis](https://redis.io) – Free in-memory database for caching and real-time features.
- [MongoDB](https://mongodb.com) – Free tier NoSQL database for flexible data models.
- [Prisma](https://prisma.io) – Free next-gen ORM for Node.js and TypeScript.
- [TypeORM](https://typeorm.io) – Free ORM for TypeScript and JavaScript.

## API Tools

- [Swagger](https://swagger.io) – Free API design, docs, and testing tools.
- [Postman](https://postman.com) – Free API testing and collaboration platform.
- [Insomnia](https://insomnia.rest) – Free API client for REST and GraphQL.

## DevOps & Containers

- [Docker](https://www.docker.com) – Free containerization for consistent dev environments.
- [Kubernetes](https://kubernetes.io) – Free container orchestration for scalable apps.

## CSS & Styling

- [Tailwind CSS](https://tailwindcss.com) – Free utility-first CSS framework for rapid UIs.
- [Bootstrap](https://bootstrap.com) – Free UI toolkit for responsive websites.
- [Sass](https://sass-lang.com) – Free CSS preprocessor for more powerful styles.

## Build Tools & Compilers

- [TypeScript](https://typescriptlang.org) – Free superset of JavaScript with static types.
- [Babel](https://babeljs.io) – Free JS compiler for modern syntax support.
- [ESLint](https://eslint.org) – Free linting for consistent JavaScript code.
- [Prettier](https://prettier.io) – Free code formatter for clean code.
- [Webpack](https://webpack.js.org) – Free module bundler for web projects.
- [Rollup](https://rollupjs.org) – Free bundler optimized for libraries.
- [esbuild](https://esbuild.dev) – Free ultra-fast bundler for dev workflows.
- [Vite](https://vitejs.dev) – Free frontend tooling with fast hot reload.

## Testing & Quality

- [Storybook](https://storybook.js.org) – Free UI component playground and docs.
- [Cypress](https://cypress.io) – Free end-to-end testing for web apps.
- [Jest](https://jestjs.io) – Free JavaScript testing framework.
- [Testing Library](https://testing-library.com) – Free tools for UI testing with confidence.
- [Webpackbin](https://webpackbin.com) – Free quick module testing sandbox.
