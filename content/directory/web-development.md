## Editors & Sandboxes

- [CodeSandbox](https://codesandbox.io/) – Spin up web projects in the browser.
- [StackBlitz](https://stackblitz.com/) – Instant dev environments for JS frameworks.
- [VS Code](https://code.visualstudio.com/) – Free desktop editor with rich extensions.

## Testing & Debugging

- [Postman](https://www.postman.com/) – API testing and collections.
- [Insomnia](https://insomnia.rest/) – API client for REST/GraphQL.
- [Lighthouse](https://web.dev/measure/) – Performance and accessibility audits.

## Docs & Specs

- [Swagger Editor](https://editor.swagger.io/) – Define and test OpenAPI specs.
- [Redocly](https://redocly.com/) – Render beautiful API docs.
- [MDN Web Docs](https://developer.mozilla.org/) – Web standards reference.

## Utilities

- [Can I Use](https://caniuse.com/) – Browser support tables.
- [Regex101](https://regex101.com/) – Test regular expressions.
- [JSONLint](https://jsonlint.com/) – Validate and format JSON.
