## Design & Prototyping Tools

- [Figma](https://figma.com) – Free collaborative UI/UX design tool with powerful prototyping.
- [Penpot](https://penpot.app) – Free open-source design and prototyping for coders & teams.
- [UXFlowchart](https://uxflowchart.com) – Free UX flowchart tool for planning user flows.
- [Canva](https://canva.com) – Free design tool with templates and assets (great for quick mockups).
- [Gravit Designer](https://gravitdesigner.com) – Free vector design tool for UI and icons.
- [Vectr](https://vectr.com) – Free vector graphics editor for UI assets.
- [Adobe Express](https://adobe.com/express) – Free graphic & web asset creator.
- [GIMP](https://gimp.org) – Free open-source image editor for UI assets.
- [Krita](https://krita.org) – Free digital painting and UI asset creation tool.
- [Inkscape](https://inkscape.org) – Free vector graphic editor for illustrations & UI.

## Icons & Illustrations

- [LottieFiles](https://lottiefiles.com) – Free lightweight animations for web & apps.
- [Heroicons](https://heroicons.com) – Free UI icon set optimized for developers.
- [Feather Icons](https://feathericons.com) – Simple free icons great for coder-friendly projects.
- [Font Awesome](https://fontawesome.com) – Free icon library widely used in web dev.
- [The Noun Project](https://thenounproject.com) – Huge free icon collection (with attribution).
- [Icons8](https://icons8.com) – Free icons, illustrations, and UI assets.
- [unDraw](https://undraw.co) – Free open-source illustrations for interfaces.
- [DrawKit](https://drawkit.io) – Free illustration sets for UI and landing pages.
- [Storyset](https://storyset.com) – Free customizable scene illustrations.
- [Absurd Design](https://absurd.design) – Free surreal creative illustrations.
- [Pixeltrue Freebies](https://pixeltrue.com/freebies) – Free illustrations & design assets for startups.

## Typography

- [1001 Free Fonts](https://1001freefonts.com) – Free downloadable fonts for UI design.
- [Google Fonts](https://fonts.google.com) – Free web fonts by Google for performance-friendly typography.
- [Good Fonts](https://goodfonts.io) – Curated free web font directory.
- [Font Pair](https://fontpair.co) – Free font pairing inspiration for UI.

## Colors & Gradients

- [Coolors](https://coolors.co) – Free color palette generator for designer-coder workflows.
- [Color Hunt](https://colorhunt.co) – Curated free color palettes.
- [WebGradients](https://webgradients.com) – Free gradient backgrounds for UI themes.
- [uiGradients](https://uigradients.com) – Free UI gradient library.
- [HTML Color Codes](https://htmlcolorcodes.com) – Free color code picker and converter.

## SVG & Patterns

- [Clippy](https://clippy.design) – Free CSS clip-path maker for creative UI shapes.
- [Get Waves](https://getwaves.io) – Free SVG wave generator for modern layouts.
- [SVG Backgrounds](https://svgbackgrounds.com) – Free customizable SVG backgrounds.
- [Patternizer](https://patternizer.com) – Free pattern generator for web assets.

## Images & Photos

- [Unsplash](https://unsplash.com) – Free high-quality photos for web backgrounds.
- [Pexels](https://pexels.com) – Free stock images and videos for UI design.
- [Pixabay](https://pixabay.com) – Free images and vector graphics.

## Image Optimization

- [images.weserv.nl](https://images.weserv.nl) – Free image CDN for performance-friendly delivery.
- [Squoosh](https://squoosh.app) – Free open-source image optimizer by Google.
- [TinyPNG](https://tinypng.com) – Free image compressor for web-ready assets.
- [WebPJS](https://webpjs.org) – Free library to generate WebP images for web performance.
- [Cloudinary](https://cloudinary.com) – Free tier for image optimization and delivery.
- [ImageKit](https://imagekit.io) – Free image CDN with optimization features.
- [Responsive Breakpoints](https://responsivebreakpoints.com) – Free tool to generate responsive image sets.

## Performance & Testing

- [web.dev](https://web.dev) – Free performance best practices and design guides.
- [PageSpeed Insights](https://pagespeed.web.dev) – Free site speed & optimization analysis.
- [GTmetrix](https://gtmetrix.com) – Free performance reports with optimization suggestions.
- [BrowserStack](https://browserstack.com) – Free trial for cross-browser testing.
- [Notion Templates](https://notion.so/templates) – Free web design & UI planning templates.
